jQuery가 포함된 span태그의 테두리 파랑
→ $('span:contains("jQuery")').css('border', '5px solid blue');

jQuery가 포함되지 않은 span 태그의 테두리 빨강
→ $('span:not(:contains("jQuery"))').css('border', '5px solid red');

span을 갖는 div요소 태그의 테두리 그린
→ $('#d1 div:has(span)').css('border', '5px solid green');

아무것도 없는 빈 div 태그의 테두리 오렌지
→ $('div:empty').css('border', '5px solid orange')

div 후손 span 태그의 글자색  빨강, 크기, 굵게
→ $('#d1 div span').css({'color' : 'red', 'font-size' : '2.0em', 'font-weight' : 'bold'});
