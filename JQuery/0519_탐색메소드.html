Syntax : $(selector).next(filter)
A.next() - A의 다음 형제를 반환
A.nextAll() - A의 모든 형제를 반환
A.nextUntil(selector) - 주어진 A와 selector 사이의 다음 형제요소들을 모두 반환 (A와 selector는 포함 X)

Syntax : $(selector).prev(filter)
A.prev() -  A의 이전 형제요소를 반환
A.preveAll() - A의 모든 이전 형제요소를 반환
A.prevUntil(selector) - A와 selector 사이의 이전 형제요소를 반환 (A와 selector는 포함 X)

Syntax : $(selector).parent(filter)
A.parent() -> 부모 : A의 바로 위 부모 요소를 반환 -> closest()랑 비슷
A.pratents()-> 조상 : A의 모든 조상 요소를 반환
A.parents(selector) -> 모든 조상 중 selector인 조상
A.parentsUntil(selector) -> A와 selector 사이의 모든 조상 요소를 반환 (A와 selector는 포함 X)
