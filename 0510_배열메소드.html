indexOf : 부분 문자열의 위치를 검색한다. start는 검색 시작 위치이며 생략시 0이 적용되어 처음부터 검색한다.
lastIndexOf : 부분 문자열의 위치를 역방향에서 검색한다. start는 검색 시작 위치이며 생략 시 문자열의 제일 끝이 적용된다.
둘 다 찾는 요소가 없을 경우 -1을 리턴한다.



push(a, b, c, ...) : 배열의 끝에 추가
pop : 배열의 끝에서 제거 후 해당 값을 리턴



slice(start, end) : start ~ end-1까지 요소 추출(잘라낸다) 후 리턴
splice(index, n, a, b, c) : index번 째에서 n개를 지우고 a, b, c를 삽입

예시)
splice(2, 1, a, b, c) : 2번 째 인덱스에서 1개를 지우고 a, b, c를 삽입
splice(3, 2) : 3번 째 위치에서 2개를 삭제
splice(4, 0, a, b, c) : 4번 째 인덱스에서 0개를 지우고 a, b, c를 삽입



a.concat(b) : a 배열에 b배열이 합쳐짐 (배열병합)
b.concat(a) : b 배열에 a배열이 합쳐짐 (배열병합)

a.join() : 배열을 문자열로 변경
